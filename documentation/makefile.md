# Makefile Commands

## Setup

```bash
$ make setup
```

If you want start from scratch again use:

```bash
$ make fresh
```

## Docker

When you are setup, use the following commands to manage your existing setup.

### Up

```bash
$ make up
```

### Down

```bash
$ make down
```

If you also want to remove the volumes use:

```bash
$ make destroy
```
